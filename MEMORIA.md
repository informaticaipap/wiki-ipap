# MEMORIA - Wiki de Procedimientos Internos IPAP

## Propósito del sitio
Wiki interna del Instituto Provincial de la Administración Pública (IPAP) de Río Negro. Centraliza procedimientos administrativos y docentes para consulta rápida del equipo informático y administrativo.

## Stack técnico
- HTML plano + Tailwind CSS v4 (CDN)
- Lucide icons (CDN)
- GitHub Pages (hosting en informaticaipap.github.io/wiki-ipap/)
- Sin frameworks JS, sin backend

## Estructura del sitio
- Header blanco con logo IPAP + "Wiki de Procedimientos"
- Sidebar con categorías colapsables + filtro
- Grid de cards con procedimientos (acordeón de pasos + botón de video)
- Footer con enlaces a sistemas (Tecnicaturas, SIU, Capacitaciones, Portal)
